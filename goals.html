Goals

The goal of this project was to build a community oriented capacity for collecting and analysing network metrics necessary for identifying internet access challenges.
The project would set up a distributed platform for collecting and analysing internet quality of service metrics within the context of a wireless community network.
The platform should be able to help detect sub-optimal or anomalous traffic flows, such as due to misconfiguration or network attacks. To achieve this, the project will build a set of distributed Internet measurement units to gather Internet performance datasets in community networks.
The outline of the project is explained in detail in the proposal embedded on this page.
